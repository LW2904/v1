Fsoc API documentation.

If you can even call that an API...

/group

/comments/:type Gets all comments from the bluestreak group, either in slim or in full format, as specified with :type.

Example URL: https://fsoc.space/api/group/comments/slim

/user

/:id Gets a steam user. :id should be of the SteamID64 type.

/:id/inventory/?appID=X&contextID=Y Gets the inventory of a steam user. Read more on appID and contextID in this post by DoctorMcKay. More information on the item properties can be found here.

Example URL: https://fsoc.space/api/user/76561198091491690

Example URL with inventory: https://fsoc.space/api/user/76561198091491690/inventory?appID=730&contextID=2
Gets the public (contextID=2) CS:GO (appID=730) inventory of user 76561198091491690.

/countries Frozen

/ Returns an object with country names as keys and the number of users as properties.

This endpoint serves a file that is being updated every 10 seconds, with data a scraper gathers. It isn't complete by far, and is more of a proof of concept and something to play with.

8/25/17 - This endpoints data is not being updated anymore, for the time being. The underlying infrastructure is being worked on. Content served is static and was gatherd over a span of about 4 hours.

Example URL: https://fsoc.space/api/countries

All endpoints are rate limited!
The default is 100 requests / 15 minutes, check the <X-RateLimit-Limit> and <X-RateLimit-Remaining> headers for information on individial endpoints.
