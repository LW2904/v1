The fsoc API. Sometimes it even works!

All enpoints are rate-limited, check the X-RateLimit-Limit and X-RateLimit-Remaining headers for exact individual limits.
/mrrobot Example

Returns a random quote from the Series Mr. Robot. Note that the quote strings are formatted and contain </br> tags.
Yes. This is edgy.

/steam/comments/:id Example

Returns an array of comments, id should be either a group (clan) or profile ID.
Note that requests to this endpoint will most likely take a while to return, since data is only cached for 12 hours.

/steam/id/:id Example

Pass a valid steam ID or vanity URL as :id parameter, returns a node-steamid object and all variations of the ID.
Note that if you don't use a vanity URL, the vanity URL of the account will be returned additionally.

/steam/user/:id Example

Pass a valid steam ID or vanity URL as :id parameter, returns a node-steamcommunity CSteamUser object.

/steam/user/:id/inventory Example

Pass a valid steam ID string (one returned by /steam/id, for example) and an appid and context as queries. Optionally, you can also pass a tradable query to also get non-tradable items. Returns an array of CEconItems. Read more on context IDs here. Note that this endpoint is heavily rate-limited, I strongly suggest you cache the data returned.

/steam/user/:id/contexts Example

Pass a valid steam ID string, returns info about what inventories (from which appIDs) are available to a user, including context IDs.

/steam/group/:id Example

Pass a valid group ID or custom group URL as :id parameter, returns a node-steamcommunity CSteamGroup object.

/steam/market/item/:appid/:hash Example

Pass an appID and market hash name, returns a node-steamcommunity CMarketItem object. This has been structured to allow direct pasting of a part of the item's market link for convenience.
Example: http://steamcommunity.com/market/listings/730/Spectrum%20Case -> https://fsoc.space/api/market/item/730/Spectrum%20Case
